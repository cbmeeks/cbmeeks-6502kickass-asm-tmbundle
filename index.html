6502 TextMate / Sublime Text 2 Bundle for KickAssember

KickAssembler is my favorite 6502 assembler for modern computers. I'm also a fan of TextMate and more recently, Sublime Text 2. Both of which can support the same bundles. These bundles allow syntax highlighting, code snippets and more.

I forgot where I got this bundle originally but it was based on the 6502 DASM bundle for TextMate. So I am slowly migrating and tweaking to better support the KickAssembler format.

Please visit the amazing KickAssembler today!

Enjoy!
